03 Flagship system
RAG / LLM ENGINEERINGCASE 01

Document intelligence that knows when not to answer.

A source-grounded QA system built around a simple rule: no relevant context, no generated answer.

PROBLEM

Generic document chatbots produce plausible answers even when the source cannot support them.

SOLUTION

A retrieval pipeline that chunks, ranks, and injects relevant evidence into Meta LLaMA through a constrained prompt.

RESULT

Predictable refusal behavior, multi-format ingestion, and a session-aware interface built for trustworthy use.

  • 01500-word chunks with 50-word overlap preserve local context
  • 02Similarity threshold prevents unsupported model responses
  • 03Complete Flask workflow moves the concept beyond a notebook
PythonFlaskMeta LLaMAOpenRouterRAG

Documents are normalized into overlapping windows, ranked against each query, and passed to the model only when retrieval quality clears a confidence threshold. Session context supports follow-up questions without weakening source-grounding.
