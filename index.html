RAG / LLM Systems Featured

RAG Chatbot with Meta LLaMA

ProblemDocument QA systems often hallucinate answers that aren't grounded in the source material.
SolutionA strict retrieval-augmented chatbot using Python, Flask, and Meta LLaMA via OpenRouter — chunking documents into 500-word segments with 50-word overlap to preserve context.
  • PDF/TXT ingestion up to 16MB using PyPDF2
  • Keyword-scored semantic search across top-3 retrieved chunks
  • Low-temperature generation tuned for factual precision
  • Strict no-hallucination policy — answers only from loaded content
PythonFlaskOpenRouter RAGLLM
Repository
Built as a full-stack document QA application with a responsive vanilla JavaScript interface and a dedicated retrieval back end. Designed to reject any query the source documents don't directly support, rather than letting the model guess.

Computer Vision / Medical AI

Pneumonia & Tuberculosis Detection with CNN

ProblemEarly respiratory-disease screening needs fast, reliable image classification, especially where specialist review is limited.
SolutionA 4-layer CNN classifying chest X-rays into Normal, Pneumonia, and Tuberculosis, paired with a desktop GUI for instant predictions.
  • 92% classification accuracy across three diagnostic classes
  • BatchNormalization, Dropout, and augmentation for generalization
  • Evaluated on accuracy, macro F1, and ROC-AUC
  • Tkinter GUI for real-time X-ray upload and confidence scoring
TensorFlowKerasCNNOpenCV
Repository
Inputs are preprocessed to 224×224 with rotation, zoom, and flip augmentation. The project demonstrates a practical, deployable computer-vision workflow for healthcare screening rather than a notebook-only benchmark.
