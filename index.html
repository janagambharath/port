FEATURED / 01RAG · LLM SYSTEMS

Grounded document intelligence with Meta LLaMA

01

A strict retrieval-augmented QA system designed to answer from source material—not model memory.

THE PROBLEM

Document assistants often produce fluent but unsupported answers, making them difficult to trust.

THE BUILD

A Flask application that chunks, embeds, retrieves, and injects relevant context into Meta LLaMA through OpenRouter.

  • 01Strict no-context / no-answer behavior to reduce hallucination risk
  • 02500-word chunks with 50-word overlap for context continuity
  • 03Multi-format document ingestion with session-aware interaction
PythonFlaskMeta LLaMAOpenRouterRAG
View repository ↗
Architecture notes +

Files are normalized into overlapping text windows, ranked against the user query, and passed to the model under a constrained prompt. The interface preserves session context while keeping source-grounding as the governing behavior.

02chunks = retrieve(query, k=4)

04if context.score < threshold:
05return NO_CONTEXT
06answer = llama(context)
